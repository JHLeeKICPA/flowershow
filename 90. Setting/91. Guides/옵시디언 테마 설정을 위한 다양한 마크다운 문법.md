## Callouts (Obsidian-specific)

> [!example] This is a note
> Notes help to emphasize information

> [!quote] Tip
> Use callouts for better readability!

> [!quote] 명언 모음집
> 옵시디언 노트에 취향과 관점을 담아 기록하면 이 노트들이 나에게 말을 걸어오기 시작한다.
> — 구요한

노트의 연결


Syntax:
```
> [!note] This is a note
> Notes help to emphasize information

> [!tip] Tip
> Use callouts for better readability!
```
